# Research Lab Manager Database System

This project implements a comprehensive database system designed to manage
research labs in a university environment. The system tracks lab members,
projects, equipment, grants, publications, and mentorship relationships.

The project follows a complete database design lifecycle:
- Conceptual design (EER)
- Logical design (Relational Schema)
- Physical implementation (SQL + application)

## Project Context
- Course: CS 631 – Data Management Systems Design
- Topic: Research Lab Manager
- Database Model: Relational
- Status: Deliverables 1 and 2 completed; Deliverable 3 in progress

## Repository Structure
- `docs/` – Project documentation and design explanations
- `sql/` – Database schema, data population, and query scripts
- `app/` – Menu-driven application (Deliverable 3)
- `assets/` – Diagrams and images used in documentation
